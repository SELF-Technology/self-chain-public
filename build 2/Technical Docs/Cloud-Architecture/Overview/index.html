How to Build Apps for SELF Usersâ€‹

1. User Authenticationâ€‹

Users log in with their SELF account:

  appId: 'your-app-id',
  permissions: ['ai:chat', 'storage:read']
});

2. Interact with User's Private LLMâ€‹

  message: "Help me plan my day",
  context: "productivity"
});

3. Store in User's Sovereign Storageâ€‹

// Save to user's private storage (with permission)

  value: { preferences: {...} },
  encrypted: true
