2. Email Protection Test

The email link has been converted to use JavaScript to prevent Cloudflare from injecting its script.

Original (Triggers Cloudflare)

<a href="mailto:info@self.app">Contact Us</a>

New (Bypasses Cloudflare)

<a href="#" onclick="window.location.href='mailto:' + 'info' + '@' + 'self.app'; return false;">Contact Us</a>
